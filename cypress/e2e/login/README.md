## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/sammieaudu007-ctrl/Cypress-automation.git
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Tests

Open Cypress Test Runner:

```bash
npx cypress open
```

Run tests in headless mode:

```bash
npx cypress run
```
